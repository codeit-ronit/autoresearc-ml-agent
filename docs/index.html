The AI Agent Team

All 4 agents are powered by Claude Opus with extended thinking.

🔬

Lead Researcher

Analyzes all past experiments, reasons about causal mechanisms, and proposes exactly ONE hypothesis per iteration with expected impact.

💻

Code Agent

Translates hypotheses into surgical code edits using find-and-replace markers. Never rewrites the whole file — eliminates LLM truncation bugs.

🛡

Auditor

Validates every edit against hard constraints. Rejects unsafe changes and feeds rejection reasons back for self-correction (up to 3 retries).

🧬

Belief Distiller

After 3+ failures of a pattern, auto-writes FORBIDDEN rules into persistent memory. Synthesizes causal beliefs from experiment history.

Autonomous Scientific Discoveries

The Belief Distiller synthesized these insights without any human guidance.

Proven Insight
Hyperparameter changes form causal chains, not independent improvements.

LR halving failed at iter 6 but succeeded at iter 19 — only after batch halving at iter 14. Grad clip tightening (iter 7) enabled batch halving by stabilizing gradients against increased noise.

Proven Insight
The model is data/signal-bottlenecked, not capacity-bottlenecked.

Both increasing capacity (D_MODEL 128→192, N_LAYERS 2→3) AND decreasing it (N_LAYERS 2→1) degraded accuracy. The ~82.3% ceiling reflects exhaustion of learnable signal from the data.

Proven Insight
Constant post-warmup LR dominates cosine decay under compute constraints.

With only 300 seconds of training, cosine decay wastes too many steps at near-zero LR. Tested twice in different contexts — consistently underperforms.

Auto-Forbidden (3+ failures)
Patterns the system permanently blocked:

N_LAYERS=3 (underfits in 300s) • D_FF=512 (overfits) • BATCH_SIZE=64 (fewer steps) • LR > 2e-4 (overshoots) • DROPOUT=0.2 (data-limited, not overfitting) • Label smoothing (reduces signal) • Cosine decay (wastes compute)
